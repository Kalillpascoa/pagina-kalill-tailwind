# Página Pessoal com Tailwind CSS

Este é um projeto de página pessoal desenvolvido com HTML e Tailwind CSS, com foco em design moderno, responsividade e boas práticas de desenvolvimento front-end.

## Demonstração

Acesse o site publicado:
https://kalillpascoa.github.io/pagina-kalill-tailwind/

## Tecnologias utilizadas

* HTML5
* Tailwind CSS
* Git e GitHub
* GitHub Pages
